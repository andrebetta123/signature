# Signatre

Aplicação responsável por expor endpoints de assinaturas de quadras

# Requisitos

- Java 11
- postgres

# Comandos

docker container ls

docker run --name postgres -e POSTGRES_PASSWORD=123456 -d -p 5432:5432 postgres:14.0

docker start postgres
